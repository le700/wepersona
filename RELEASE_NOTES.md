# WePersona 2.0 正式发布

## 🎉 介绍

WePersona 是一个完全本地的智能社交人格平台，整合微信消息管理、数据分析、数字人格蒸馏和AI助手功能。

## ✨ 新特性

### 🔮 数字人格系统
- 四维人格定义（程序性、互动性、记忆、性格）
- 智能人格蒸馏向导
- 证据链追踪与可信度评估
- 7种预设角色模板（自己、同事、导师、亲人、伴侣、朋友、公众人物）
- 人格版本管理与快照

### 🤖 AI 聊天助手
- 支持多种 AI 模型（DeepSeek、OpenAI、Claude、通义千问、豆包、科大讯飞等）
- 人格上下文注入
- 会话管理与历史记录
- 打字状态指示

### 📱 微信机器人
- 扫码登录
- 自动消息回复
- 白名单/黑名单配置
- 人格挂载回复
- 实时状态监控

### 📊 数据分析可视化
- 消息热力图
- 词云分析
- 日历视图（类似 GitHub 贡献图）
- 时段分布统计
- 互动频率分析

## 📦 功能继承

WePersona 完全继承了 WeFlow 的所有功能：

- 💬 本地实时查看聊天记录
- 📸 朋友圈图片、视频、实况的预览和解密
- 📈 统计分析与群聊画像
- 📊 年度报告与可视化概览
- 📤 导出聊天记录为 HTML 等格式
- 🛡️ 消息防撤回
- 🔔 实时弹窗通知
- 📋 联系人管理与导出
- 🔌 HTTP API 接口

## 🎨 界面更新

- 专业的深色主题 UI
- 全新的数字人格管理界面
- AI 聊天界面优化
- 微信机器人控制面板
- 数据分析可视化面板

## 🛠️ 技术栈

- **框架**：Electron + React 19 + TypeScript
- **状态管理**：Zustand + Persist
- **构建工具**：Vite
- **可视化**：ECharts 5.4.3
- **AI 服务**：统一适配器架构
- **微信集成**：WeChaty

## 📥 下载安装

### Windows
下载 `WePersona-2.0.0-Setup.exe` 并运行安装程序

## 🚀 快速开始

1. 安装并启动 WePersona
2. 连接到你的微信数据库
3. 开始探索数字人格系统
4. 配置 AI 聊天助手
5. 启用微信机器人（可选）
