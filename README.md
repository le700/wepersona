# WePersona

WePersona 是一个**完全本地**的智能社交人格平台，整合微信消息管理、数据分析、数字人格蒸馏和AI助手功能。

---

**WePersona** is a fully local intelligent social persona platform, integrating WeChat message management, data analysis, digital persona distillation, and AI assistant features.

## 主要功能

### 🔮 数字人格系统
- 四维人格定义（程序性、互动性、记忆、性格）
- 智能人格蒸馏向导
- 证据链追踪与可信度评估
- 7种预设角色模板
- 人格版本管理与快照

### 🤖 AI 聊天助手
- 支持多种 AI 模型（DeepSeek、OpenAI、Claude、通义千问、豆包等）
- 人格上下文注入
- 会话管理与历史记录
- 打字状态指示

### 📱 微信机器人
- 扫码登录
- 自动消息回复
- 白名单/黑名单配置
- 人格挂载回复
- 实时状态监控

### 📊 数据分析可视化
- 消息热力图
- 词云分析
- 日历视图
- 时段分布统计
- 互动频率分析

### 💬 微信消息管理（继承自 WeFlow）
- 本地实时查看聊天记录
- 朋友圈图片、视频、实况的预览和解密
- 统计分析与群聊画像
- 年度报告与可视化概览
- 导出聊天记录为 HTML 等格式
- 消息防撤回
- 实时弹窗通知

---

**Key Features**

### 🔮 Digital Persona System
- Four-dimensional persona definition (Procedural, Interactive, Memory, Personality)
- Intelligent persona distillation wizard
- Evidence chain tracking and credibility assessment
- 7 preset role templates
- Persona version management and snapshots

### 🤖 AI Chat Assistant
- Support for multiple AI models (DeepSeek, OpenAI, Claude, Tongyi Qianwen, Doubao, etc.)
- Persona context injection
- Session management and history
- Typing status indicator

### 📱 WeChat Bot
- QR code login
- Auto message reply
- Whitelist/blacklist configuration
- Persona-mounted replies
- Real-time status monitoring

### 📊 Data Analysis & Visualization
- Message heatmap
- Word cloud analysis
- Calendar view
- Time period distribution statistics
- Interaction frequency analysis

### 💬 WeChat Message Management (inherited from WeFlow)
- Local real-time chat history viewing
- Preview and decrypt Moments photos, videos, and Live Photos
- Statistical analysis and group chat insights
- Annual reports and visual overviews
- Export chat history to HTML and other formats
- Anti-recall protection
- Real-time popup notifications

## 详细功能清单

| 功能模块 | 说明 |
|---------|------|
| **数字人格管理** | 完整的人格创建、编辑、删除、选择功能，四维维度定义，证据追踪系统 |
| **人格蒸馏向导** | 从微信聊天记录中智能提取人格特征，多维度数据采集与分析 |
| **AI 聊天** | 支持多模型对话，人格上下文注入，会话历史管理 |

| **消息防撤回** | 防止其他人发送的消息被撤回 |
| **实时弹窗通知** | 新消息到达时提供桌面弹窗提醒，黑白名单功能 |
| **私聊分析** | 统计好友间消息数量，分析消息类型与发送比例，查看消息时段分布 |
| **群聊分析** | 查看群成员详细信息，分析群内发言排行、活跃时段和媒体内容 |
| **年度报告** | 生成按年统计的年度报告，或跨年度的长期历史报告 |
| **双人报告** | 选择指定好友，基于双方聊天记录生成专属分析报告 |
| **消息导出** | 将微信聊天记录导出为多种格式：JSON、HTML、TXT、Excel、CSV 等 |
| **朋友圈** | 解密朋友圈图片、视频、实况；导出朋友圈内容；拦截删除与隐藏操作 |
| **联系人** | 导出微信好友、群聊、公众号信息 |
| **热力图可视化** | 消息时间分布热力图，直观展示聊天活跃时段 |
| **词云分析** | 提取关键词生成词云，快速识别聊天热点话题 |
| **日历视图** | 按日期展示消息密度，类似 GitHub 贡献图 |

**Detailed Feature List**

| Feature Module | Description |
|----------------|-------------|
| **Digital Persona Management** | Complete persona creation, editing, deletion, and selection; four-dimensional definition; evidence tracking system |

## 面向开发者

如果你想从源码构建或为项目贡献代码，请遵循以下步骤：

```bash
# 1. 克隆项目到本地
git clone https://github.com/le700/wepersona.git
cd wepersona

# 2. 安装项目依赖
npm install --legacy-peer-deps

# 3. 运行应用（开发模式）
npm run dev

# 4. 构建应用
npm run build:frontend
```

---

**For Developers**

If you want to build from source or contribute code to the project, please follow these steps:

```bash
# 1. Clone the project locally
git clone https://github.com/le700/wepersona.git
cd wepersona

# 2. Install project dependencies
npm install --legacy-peer-deps

# 3. Run the application (development mode)
npm run dev

# 4. Build the application
npm run build:frontend
```

## 技术架构

- **框架**：Electron + React 19 + TypeScript
- **状态管理**：Zustand + Persist
- **构建工具**：Vite
- **可视化**：ECharts 5.4.3
- **AI 服务**：统一适配器架构（DeepSeek、OpenAI、Claude、通义千问、豆包等）
- **微信集成**：WeChaty

---

**Technical Architecture**

- **Framework**：Electron + React 19 + TypeScript
- **State Management**：Zustand + Persist
- **Build Tool**：Vite
- **Visualization**：ECharts 5.4.3
- **AI Service**：Unified adapter architecture (DeepSeek, OpenAI, Claude, Tongyi Qianwen, Doubao, etc.)
- **WeChat Integration**：WeChaty
